# 陌遇 App - 功能特性详解

## 🎯 核心功能

### 1. 首页 - 探索发现 (Discovery)

#### 1.1 Hero 图片展示
- **功能描述**: 全屏沉浸式首图，展示精选旅行目的地
- **交互方式**: 向下滚动时产生视差效果
- **视觉效果**: 渐变遮罩层，突出标题文字
- **技术实现**: Stack + Image.network + 渐变Container

#### 1.2 悬浮搜索栏
- **功能描述**: 毛玻璃效果的搜索输入框
- **位置**: 固定在Hero图片顶部
- **视觉效果**: 半透明背景，白色边框
- **交互**: 点击进入搜索页面（预留）

#### 1.3 分类导航
- **功能描述**: 四个主要旅行分类快捷入口
- **分类**: 自然、户外、文化、娱乐
- **视觉**: 彩色图标 + 圆角卡片
- **交互**: 点击进入对应分类页面（预留）

#### 1.4 探索手记
- **功能描述**: 用户UGC内容展示
- **布局**: 2列网格布局
- **内容**: 标题、作者、图片、点赞数
- **交互**: 点击查看详情（预留）

#### 1.5 出行攻略
- **功能描述**: 旅行攻略推荐
- **布局**: 横向滚动卡片
- **内容**: 标题、标签、阅读时间
- **视觉**: 白色卡片 + 粉色标签

#### 1.6 奇遇指南
- **功能描述**: 随机推荐功能入口
- **视觉**: 渐变色背景 + CTA按钮
- **交互**: 点击获取随机推荐
- **特效**: 装饰性圆形元素

---

### 2. 漫游 - 地点探索 (Roaming)

#### 2.1 卡片堆叠
- **功能描述**: Tinder风格的地点浏览
- **视觉**: 主卡片 + 背景卡片
- **交互**: 滑动切换下一个地点
- **动画**: 卡片缩放和位移效果

#### 2.2 分类切换
- **功能描述**: 四个Tab分类
- **分类**: 景点、美食、住宿、打卡
- **视觉**: 圆角胶囊式Tab栏
- **交互**: 点击切换分类

#### 2.3 地点信息
- **展示内容**:
  - 地点名称（大标题）
  - 位置信息（图标+文字）
  - 距离显示（顶部标签）
  - 标签展示（底部）
- **视觉**: 渐变遮罩 + 白色文字

#### 2.4 操作按钮
- **收藏按钮**: 爱心图标，点击收藏
- **分享按钮**: 发送图标，点击分享
- **视觉**: 毛玻璃圆形按钮
- **位置**: 右上角垂直排列

---

### 3. AI助手 - 智能规划 (AI)

#### 3.1 AI球体
- **功能描述**: 动态呼吸的AI图标
- **视觉**: 渐变色圆形 + 机器人图标
- **动画**: 脉冲缩放效果
- **技术**: AnimationController + ScaleTransition

#### 3.2 预设问题
- **功能描述**: 四个常见问题快捷入口
- **内容**:
  - "帮我规划一条 3 天的杭州路线"
  - "如果你是本地人，你会带我去哪？"
  - "寻找适合独旅的海滨地点"
  - "附近有什么不为人知的小众餐厅？"
- **视觉**: 白色卡片 + 右箭头
- **交互**: 点击发送问题

#### 3.3 输入框
- **功能描述**: 文字输入和发送
- **视觉**: 白色圆角输入框 + 粉色发送按钮
- **位置**: 底部固定
- **交互**: 输入文字后点击发送

#### 3.4 背景装饰
- **功能描述**: 渐变色装饰球
- **视觉**: 模糊的彩色圆形
- **效果**: 营造梦幻氛围

## 🎨 设计特性

### 视觉风格

#### Glassmorphism
- **毛玻璃背景**: 半透明 + 模糊效果
- **应用场景**: 导航栏、搜索栏、卡片
- **技术实现**: BackdropFilter + blur

#### 圆角设计
- **标准圆角**: 24px
- **小圆角**: 16px
- **大圆角**: 32px
- **应用**: 所有卡片和按钮

#### 阴影效果
- **彩色阴影**: 粉色半透明
- **模糊半径**: 20-40px
- **偏移**: (0, 10)
- **应用**: 主要卡片和按钮

### 动画效果

#### 页面切换
- **类型**: 淡入淡出
- **时长**: 300ms
- **曲线**: easeInOut

#### 导航动画
- **类型**: 缩放 + 位移
- **时长**: 300ms
- **效果**: 选中态上浮

#### AI球体
- **类型**: 脉冲缩放
- **时长**: 2000ms
- **循环**: 无限循环
- **范围**: 0.95 - 1.05

#### 卡片切换
- **类型**: 滑动 + 缩放
- **时长**: 400ms
- **效果**: 流畅过渡
