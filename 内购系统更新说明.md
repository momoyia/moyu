## 使用说明

### 1. 在 App Store Connect 配置内购商品
确保在 App Store Connect 中添加了所有 13 个内购商品 ID：
- moyuiapid_1 到 moyuiapid_13

### 2. 测试内购
1. 在 Xcode 中运行应用
2. 进入"设置" → "金币商店"
3. 选择任意商品点击"购买"
4. 使用沙盒测试账号完成购买

### 3. 查看金币余额
- 金币余额显示在商店页面顶部
- 购买成功后自动增加
- 数据持久化保存

## 技术细节

### 金币存储
```dart
// 保存金币
final prefs = await SharedPreferences.getInstance();
await prefs.setInt('coinBalance', _coinBalance);

// 读取金币
int coins = prefs.getInt('coinBalance') ?? 600;
```

### 内购流程
1. 用户点击"购买"按钮
2. 调用 `_handlePurchase(bundle)`
3. 从 `_productDetails` 获取商品信息
4. 调用 `_purchaseManager.initiateTransaction(product)`
5. iOS 系统弹出购买确认
6. 购买成功后回调 `_handlePurchaseSuccess(coins)`
7. 更新金币余额并保存

### 价格显示
- 优先显示从 App Store 获取的真实价格
- 如果未加载，显示配置的默认价格
- 格式：`product?.price ?? bundle.price`

## 注意事项

1. **不要修改内购流程**：`LinstPurchaseManager` 中的逻辑已经正确，不要改动
2. **ID 必须匹配**：确保 `LinstBundle` 中的 `itemId` 与 App Store Connect 中的完全一致
3. **测试环境**：使用沙盒测试账号测试，不会产生真实费用
4. **价格显示**：真实价格会在应用启动后从 App Store 加载
5. **小屏幕适配**：UI 已针对小屏幕优化，使用列表布局
