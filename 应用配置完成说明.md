## 🎨 图标配置详情

### pubspec.yaml 配置
```yaml
flutter_launcher_icons:
  android: true                                    # 生成 Android 图标
  ios: true                                        # 生成 iOS 图标
  image_path: "assets/images/moyu logo.png"       # 源图标路径
  adaptive_icon_background: "#FFFFFF"              # Android 自适应图标背景色（白色）
  adaptive_icon_foreground: "assets/images/moyu logo.png"  # Android 自适应图标前景
  remove_alpha_ios: true                           # iOS 移除透明通道（iOS 要求）
```

### 图标要求
- ✅ 源图标：1024x1024 或更高分辨率
- ✅ iOS：不能有透明背景（已配置 remove_alpha_ios: true）
- ✅ Android：支持透明背景和自适应图标
- ✅ 格式：PNG 格式

## 🔄 如果需要更换图标

1. 替换 `assets/images/moyu logo.png` 为新图标
2. 运行命令重新生成：
   ```bash
   flutter pub run flutter_launcher_icons
   ```
3. 清理并重新构建：
   ```bash
   flutter clean
   flutter run
   ```
