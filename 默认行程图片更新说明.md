# 默认行程图片更新说明

## 问题说明

默认行程的图片路径已经在代码中更新为：
```dart
'imageUrl': 'assets/images/outside/johannes-andersson-UCd78vfC8vU-unsplash.jpg'
```

但是如果应用之前已经运行过，默认行程数据已经保存到 SharedPreferences 中，新的配置不会自动生效。

## 解决方案

### 方案一：清除应用数据（推荐）

#### iOS 模拟器
```bash
# 完全卸载应用
flutter clean
# 重新安装
flutter run -d iphone
```

#### Android 模拟器/真机
```bash
# 清除应用数据
adb shell pm clear com.example.moyu_app
# 或者在设置中手动清除应用数据
```

### 方案二：在设置页面添加"重置数据"功能

可以在设置页面添加一个"重置数据"按钮，清除所有本地数据：

```dart
Future<void> _resetData() async {
  final prefs = await SharedPreferences.getInstance();
  await prefs.clear();
  // 重启应用
}
```

### 方案三：修改默认行程 ID

如果不想清除所有数据，可以修改默认行程的 ID，这样会创建一个新的默认行程：

在 `lib/services/storage_service.dart` 中：
```dart
final defaultItinerary = {
  'id': 'default_2',  // 改成 default_2
  'title': '云南大理之旅',
  // ... 其他配置
};
```

5. 查看"我的行程"中的默认行程
6. 应该显示新的图片

## 注意事项

- 默认行程只在首次使用时创建
- 如果用户已经有行程数据，不会自动更新
- 清除数据会删除所有用户数据（收藏、行程、设置等）
