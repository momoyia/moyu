# 头像上传功能说明

## 已完成的功能

### 1. 添加了图片选择依赖
- 在 `pubspec.yaml` 中添加了 `image_picker: ^1.0.7`

### 2. 配置了 iOS 权限
- 在 `ios/Runner/Info.plist` 中添加了相册和相机权限说明：
  - `NSPhotoLibraryUsageDescription`: 需要访问您的相册以选择头像
  - `NSCameraUsageDescription`: 需要访问您的相机以拍摄头像

### 3. 修改了 ProfileScreen
- 添加了 `ImagePicker` 实例
- 添加了 `_avatarPath` 状态变量来存储选中的头像路径
- 实现了 `_pickAvatar()` 方法来调起相册选择图片
- 在头像右下角添加了相机图标按钮
- 头像显示逻辑：
  - 如果用户选择了本地图片，显示本地图片
  - 否则显示默认的网络头像
  - 如果加载失败，显示默认图标

### 4. UI 设计
- 相机图标位于头像右下角
- 图标样式：
  - 粉色圆形背景 (AppTheme.primaryPink)
  - 白色边框
  - 白色相机图标
  - 带阴影效果
  - 尺寸：28x28

## 使用方法

1. 打开应用，进入"我的"页面
2. 点击头像右下角的相机图标
3. 系统会弹出相册选择器
4. 选择一张图片作为头像
5. 头像会立即更新并保存到本地

## 技术实现

- 使用 `image_picker` 包调起系统相册
- 使用 `shared_preferences` 保存头像路径
- 图片会被压缩到 800x800，质量 85%
- 支持 iOS 和 Android 平台

## 注意事项

- 头像文件存储在应用沙盒中
- 卸载应用后头像会丢失
- 如需持久化存储，建议上传到服务器
