# 青少年模式功能说明

## 功能概述

青少年模式提供内容过滤和购买保护功能，家长可以设置密码来控制孩子的购买行为。

## 主要功能

### 1. 青少年模式开关
- 开启后自动过滤不适宜内容
- 可以随时开启或关闭
- 设置密码后，关闭模式需要验证密码

### 2. 密码保护
- 支持设置6位数字密码
- 密码用于保护以下操作：
  - 关闭青少年模式
  - 进行购买（当开启购买限制时）
- 支持修改密码
- 支持移除密码

### 3. 购买限制
- 开启后，购买金币需要输入密码
- 必须先设置密码才能开启购买限制
- 有效防止未成年人未经授权的消费

## 使用流程

### 首次设置

1. **进入青少年模式设置**
   - 打开"设置"
   - 点击"安全模式" → "青少年模式"

2. **开启青少年模式**
   - 打开"启用青少年模式"开关
   - 系统开始过滤不适宜内容

3. **设置密码（可选）**
   - 点击"设置密码"
   - 输入6位数字密码
   - 确认密码
   - 密码设置成功

4. **开启购买限制（可选）**
   - 设置密码后，可以开启"限制购买"
   - 开启后，购买时需要输入密码

### 购买流程（开启限制后）

1. 用户进入金币商店
2. 点击任意商品的"购买"按钮
3. 系统弹出密码验证对话框
4. 输入正确密码后才能继续购买
5. 密码错误则无法购买

### 修改密码

1. 进入青少年模式设置
2. 点击"修改密码"
3. 输入当前密码验证
4. 输入新密码
5. 确认新密码

### 移除密码

1. 进入青少年模式设置
2. 点击"移除密码"
3. 输入当前密码验证
4. 确认移除
5. 密码保护已移除

### 关闭青少年模式

1. 进入青少年模式设置
2. 关闭"启用青少年模式"开关
3. 如果设置了密码，需要输入密码验证
4. 验证成功后关闭青少年模式

## 数据存储

所有设置保存在 SharedPreferences 中：

| 键名 | 类型 | 说明 |
|------|------|------|
| youth_mode | bool | 青少年模式是否开启 |
| youth_mode_password | String | 密码（6位数字） |
| youth_mode_restrict_purchase | bool | 是否限制购买 |

## 安全特性

### 1. 密码验证
- 密码必须是6位数字
- 两次输入必须一致
- 密码错误时显示提示
- 密码正确才能继续操作

### 2. 操作保护
- 关闭青少年模式需要密码
- 购买金币需要密码（开启限制时）
- 修改密码需要验证旧密码
- 移除密码需要验证密码

### 3. 数据持久化
- 所有设置自动保存
- 应用重启后设置保持
- 密码加密存储（建议后续升级）

## 后续优化建议

1. **密码加密**
   - 使用 SHA-256 加密存储密码
   - 不存储明文密码

2. **密码找回**
   - 添加密保问题
   - 或绑定家长手机号

3. **使用时长限制**
   - 设置每日使用时长
   - 超时自动锁定

4. **内容过滤**
   - 实现真实的内容过滤逻辑
   - 根据年龄分级

5. **家长监控**
   - 查看使用记录
   - 查看消费记录
   - 远程管理设置

6. **生物识别**
   - 支持指纹验证
   - 支持面容识别

## 注意事项

1. 密码必须妥善保管，忘记密码需要重新设置
2. 移除密码会同时关闭购买限制
3. 关闭青少年模式不会删除密码
4. 密码仅在本地存储，不会上传服务器
5. 建议定期更换密码以提高安全性
