# 启动页和登录功能说明

## 已完成的功能

### 1. 启动页 (SplashScreen)
**文件**: `lib/screens/splash_screen.dart`

**功能**:
- 显示陌遇 Logo（使用本地资源 `assets/images/moyu logo.png`）
- 停留 2 秒后自动跳转
- 检查登录状态：
  - 已登录 → 跳转到主页面
  - 未登录 → 跳转到登录页

### 2. 登录页 (LoginScreen)
**文件**: `lib/screens/login_screen.dart`

**UI 元素**:
- Logo 图标（120x120）
- 应用名称："陌遇"
- Slogan："遇见世界，遇见更好的自己"
- 立即登录按钮（粉色主题）
- 用户协议勾选框

**功能**:
- 协议勾选验证：
  - 未勾选时点击登录 → 弹出温馨提示对话框
  - 已勾选时点击登录 → 保存登录状态并跳转到主页
- 可点击《用户协议》和《隐私政策》查看详情
- 使用 RichText 和 TapGestureRecognizer 实现文字链接

### 3. 用户协议页面 (UserAgreementScreen)
**文件**: `lib/screens/user_agreement_screen.dart`

**内容包括**:
- 协议的接受
- 账号注册
- 用户行为规范
- 内容版权
- 服务的变更、中断或终止
- 免责声明
- 协议的修改
- 联系方式

### 4. 隐私政策页面 (PrivacyPolicyScreen)
**文件**: `lib/screens/privacy_policy_screen.dart`

**内容包括**:
- 引言
- 我们收集的信息
- 我们如何使用信息
- 信息的存储
- 信息的共享
- 您的权利
- 未成年人保护
- 隐私政策的变更
- 联系方式

### 5. 退出登录功能
**位置**: 设置页面底部

**功能**:
- 红色边框按钮
- 点击后弹出确认对话框
- 确认后清除登录状态并返回登录页

## 使用流程

### 首次使用
1. 打开应用 → 显示启动页（2秒）
2. 自动跳转到登录页
3. 阅读并勾选用户协议和隐私政策
4. 点击"立即登录"按钮
5. 进入主页面

### 再次打开
1. 打开应用 → 显示启动页（2秒）
2. 检测到已登录状态
3. 直接进入主页面

### 退出登录
1. 进入"我的"页面
2. 点击右上角设置图标
3. 滚动到底部
4. 点击"退出登录"按钮
5. 确认后返回登录页

## 技术实现

### 登录状态管理
- 使用 `shared_preferences` 存储登录状态
- Key: `is_logged_in`
- Value: `true` (已登录) / `false` (未登录)

### 页面导航
- 启动页使用 `pushReplacement` 替换当前页面
- 登录成功使用 `pushReplacement` 防止返回
- 退出登录使用 `pushAndRemoveUntil` 清空导航栈

### 协议勾选
- 使用 `bool _agreedToTerms` 状态变量
- 自定义勾选框 UI
- 未勾选时显示温馨提示对话框

### 文字链接
- 使用 `RichText` 组件
- 使用 `TapGestureRecognizer` 实现点击事件
- 点击后使用 `Navigator.push` 跳转到对应页面

## 注意事项

1. Logo 图片路径：`assets/images/moyu logo.png`
2. 登录功能为无账号登录，仅保存登录状态
3. 协议和隐私政策内容为示例，实际使用时需要根据真实情况修改
4. 退出登录会清除登录状态，但不会清除其他用户数据（如收藏、行程等）
