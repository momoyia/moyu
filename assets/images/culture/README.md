# 文化图片资源

这个文件夹包含了用于出行攻略和好玩指南的文化主题图片。

## 图片使用

- **出行攻略**: 周末citywalk、说走就走、治愈系小城
- **好玩指南**: 西湖、大理古城、鼓浪屿、外滩
- **相关游记**: 旅行攻略详情页中的相关游记封面

## 图片分配

图片通过哈希算法随机分配，确保：
- 每个地点/攻略都有唯一的图片
- 不会出现重复的图片
- 图片分配是确定性的（相同的名称总是得到相同的图片）

## 压缩说明

这些图片已经通过压缩脚本进行了优化：
- JPEG 质量: 85%
- 最大尺寸: 1200x1200
- 移除了元数据
- 保持了视觉质量的同时减小了文件体积
