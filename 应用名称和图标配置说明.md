## 需要手动完成的图标配置

由于 Flutter 应用图标需要多种尺寸，建议使用 `flutter_launcher_icons` 包来自动生成。

### 方法一：使用 flutter_launcher_icons（推荐）

#### 1. 添加依赖
在 `pubspec.yaml` 中添加：

```yaml
dev_dependencies:
  flutter_launcher_icons: ^0.13.1

flutter_launcher_icons:
  android: true
  ios: true
  image_path: "assets/images/moyu logo.png"
  adaptive_icon_background: "#FFFFFF"
  adaptive_icon_foreground: "assets/images/moyu logo.png"
```

#### 2. 运行命令生成图标
```bash
flutter pub get
flutter pub run flutter_launcher_icons
```

这将自动生成所有需要的图标尺寸并放置到正确的位置。

### 方法二：手动替换图标

如果要手动替换，需要准备以下尺寸的图标：

#### iOS 图标尺寸（放在 `ios/Runner/Assets.xcassets/AppIcon.appiconset/`）
- 20x20 @1x, @2x, @3x
- 29x29 @1x, @2x, @3x
- 40x40 @1x, @2x, @3x
- 60x60 @2x, @3x
- 76x76 @1x, @2x
- 83.5x83.5 @2x
- 1024x1024 @1x

#### Android 图标尺寸
放在对应的文件夹中：
- `android/app/src/main/res/mipmap-mdpi/` - 48x48
- `android/app/src/main/res/mipmap-hdpi/` - 72x72
- `android/app/src/main/res/mipmap-xhdpi/` - 96x96
- `android/app/src/main/res/mipmap-xxhdpi/` - 144x144
- `android/app/src/main/res/mipmap-xxxhdpi/` - 192x192

### 方法三：使用在线工具

可以使用以下在线工具生成图标：
1. https://www.appicon.co/
2. https://icon.kitchen/
3. https://makeappicon.com/

上传 `assets/images/moyu logo.png`，下载生成的图标包，然后替换到对应目录。

## 注意事项

1. **图标要求**：
   - 建议使用 1024x1024 的高清图标作为源文件
   - iOS 图标不能有透明背景
   - Android 可以使用透明背景（Adaptive Icon）

2. **应用名称长度**：
   - iOS：建议不超过 12 个字符
   - Android：建议不超过 15 个字符
   - "陌遇" 只有 2 个字符，完全符合要求

3. **重新安装**：
   - 修改图标后需要完全卸载应用重新安装才能看到新图标
   - 或者使用 `flutter clean` 清理后重新构建
